# Bank Transaction Ledger SQL Project

## Project Overview

This project demonstrates SQL skills using a simulated Bank Transaction Ledger database. It includes realistic banking data and SQL queries ranging from beginner to advanced levels.

## Database

The project contains three tables:

- Customers
- Accounts
- Bank Transactions

## Dataset

- 50 Customers
- 50 Accounts
- 500 Transactions

## SQL Concepts Covered

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions
- INNER JOIN
- CASE
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- RANK()
- DENSE_RANK()
- ROW_NUMBER()
- LAG()
- LEAD()
- Running Total

## Example Business Questions

- Find the top customers by transaction amount.
- Calculate total deposits by branch.
- Identify customers with the highest account balances.
- Generate running balances for each account.
- Analyze transaction patterns using window functions.
